Projects

CareSyncAI

As part of a course project in collaboration with Daresay, my team and I designed and implemented a native application for elderly home care. The purpose of the application is to facilitate communication and planning of shifts between the staff and management. One main function in the application is an AI feature that auto-generates a summary from the latest documentation about the elderly.

StoryAI

Throughout a course project, my team and I designed and developed an application that utilized AI to automatically generate children's stories. The story's content was based on user inputs, including character details, environments, and plot preferences. A fun project where the biggest challenge was to implement and interact with OpenAI's API to optimize the story based on the user's preferences.

Music Quiz

As a part of a course project, me and my team designed a function in collaboration with a music streaming service in Sweden. The function was for an application and the purpose was to enable the users to create, host and join interactive music quizzes. A key feature is a generative AI option that assists the quiz creator in generating questions based on the music in the participants' playlists.

Pepper Robot

In a Human-Robot Interaction course, our group of three designed and implemented the Pepper robot to serve as a sales assistant in a cafe using both movments, verbal audio and an interactive interface that enabled the customer to interact with the robot to order food. The implementation was conducted in Choregraphe, and the UI design was created in Figma. It was a challenging and fun project, as it was the first time any of us had implemented a robot. However, when we got it to work, it was exciting to see the robot perform the tasks we had programmed.
