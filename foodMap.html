Food Map


As a course project, a classmate and I created a React web application called Food Map. The application uses two APIs to allow users to search for locations, scroll through a list of all the restaurants within the visible map area, and navigate the map using the pointer.

Task

Our task was to create a website by integrating and combining two different APIs. We were free to use any framework we preferred, with the focus placed on functionality and API integration rather than the design.


Concept idea

We came up with the idea to use a map API and combine it with an API that handles restaurants. Using these two, the user should be able to find restaurants given the location from the map.

Process

My group mate and I collaborated using pair programming. This allowed us to work efficiently and review each other's code for quality.

The challenge was planning how the two systems should interact with each other. For example, it was difficult to figure out how to retrieve all the restaurants within the map area because we initially only used one specific location on the map. We later solved this by taking the corners of the map and fetching all the restaurants within that area.

Result

We successfully integrated the two APIs, enabling the website to function properly. The website allowed users to search for locations, scroll through a list of all the restaurants within the visible map area, and navigate the map using the pointer. Additionally, users could create an account to save their favorite restaurants for future reference.
